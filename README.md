# Radiostations  

This repo contains a list of (radio)streams  
This is also the place for releases of the android app:  
***Just An Audio Player***  

## What is it  

***Just An Audio Player*** is an app for android that plays a network audio stream. You can add all kinds of radio streams from the list in this repo. Or you can add a custom stream.

## How to use the app  

Click on an item in the main list to select a stream. Tap the *PLAY* button on the bottom to start playing the stream.  
To **add a stream** tap on the logo and the list of streams will popup. Tap on a stream to add it to your list.  
Tap on *CUSTOM STREAM* to add a stream not in the list.  
To **change a stream** long press on an item from the main list. A popup will open with the stream info. Here you can make the required changes and also set the stream as favorite by clicking the heart on top. Do not forget to tap *CHANGE* to implement the changes.  
The popup also provides a button *DELETE*
